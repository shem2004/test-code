Small explanations

CSS hover uses transition. Spinner uses @keyframes. Typing effect uses a simple timeout loop to append characters. Canvas uses requestAnimationFrame and basic physics for a ball bounce.

Code snippets

<!-- Typing effect JS (short) -->
const text = "Hello — welcome to tiny animations!";
let i = 0;
function type(){
  if(i < text.length) { document.getElementById('typed').textContent += text[i++]; setTimeout(type, 45); }
}
setTimeout(type, 400);
